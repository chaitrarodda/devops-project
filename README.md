#  DevOps Project

##  Project Overview
This is a simple DevOps project where a static website is containerized using Docker and deployed locally.


## Technologies Used
- Docker
- Git
- GitHub
- Linux (Ubuntu)


## Project Structure
- index.html → Website code
- Dockerfile → Container configuration


## Steps to Run the Project

1. Build Docker image
   docker build -t myproject .

2. Run container
   docker run -d -p 8080:80 myproject

3. Open in browser
   http://localhost:8080


##  Features
- Simple static website
- Containerized using Docker
- Runs locally in browser


##  Output
Website running successfully inside Docker container.
